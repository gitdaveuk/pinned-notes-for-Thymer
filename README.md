# pinned-notes-for-Thymer
Pin notes to the sidebar
<img width="298" height="239" alt="image" src="https://github.com/user-attachments/assets/f707c3e6-cb89-4ba7-ab89-ee3cfc9ff650" />
Ctrl+P "Pin note"
Once the note is pinned, you can open it normally or Ctrl+click will open it in a new panel.
Remove the pinned note by clicking on the cross next to it.
